蜜罐( Honey-pot ）： Honey-pot是一種安全技術，指在組織或企業的計算機系統、網絡環境中故意設置易受攻擊的虛擬系統或者服務，以吸引黑客前來攻擊，同時監測駭客威脅並防止對内部系統的真實攻擊。 honey-pot is a security tech that intentionally places vulnerable virtual systems or services in an enterprise or organization’s computer system or network environment to attract hackers to attack, while monitoring hacker threats and preventing real attacks on internal systems. 蜜罐技術是一種主動的安全防範技術，通過誘捕攻擊者，獲取攻擊者的信息及攻擊手段，從而達到預防和追蹤的效果。 Honey-pot technology is an active security measure that capture attackers to get the information and attack methods of attackers, thereby better protecting their own computer system and network environment and tracking attackers....
